# Dashboard-Instacart
Projeto pessoal usando Streamlit e Plotly para explorar dados públicos do Instacart (2018).
---
## Estrutura
```
.
├── instacart_dashboard.py
├── orders.csv
├── order_products.csv
├── products.csv
├── departments.csv
├── aisles.csv
├── requirements.txt
├── notebooks
│   └── EDA.ipynb
└── .streamlit
    └── config.toml
```
---
## Como rodar
```bash
pip install -r requirements.txt
streamlit run instacart_dashboard.py
```
